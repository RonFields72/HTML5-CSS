Position Property

  • static: Elements display in the order they are listed in the HTML. This is the default.
  • absolute: Element is display relative to the first parent element that can be found that is NOT static. If not found, then the <html> tag is the containing parent.
  • fixed: Element is displayed relative to the browser window. Does not move even if window is scrolled.
  • relative: Element is displayed relative to its normal position. If 'left:2em' is specified then 2 em's are added to the element's normal left position.
