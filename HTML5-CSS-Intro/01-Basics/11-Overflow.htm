Overflow Property

The overflow property specifies what to do if the content of an element exceeds the size of an element's box.

  • visible: Render the additional text outside the element
  • hidden: Clip the additional text and hide the text
  • scroll: Add scroll bars to element to see the rest of the text
  • auto: If additional text is clipped, scroll bars should be added, but this is left up to the browser.

overflow:hidden

You will use the overflow property when you have a large amount of text that needs to fit into a certain size element and you are not sure if it will fit or not.
