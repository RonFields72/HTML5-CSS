Font Properties

  • Font-Family: generic and specific font
    • generic: Serif, Sans-Serif, Monospace
    • font: Times New Roman, Georgia, Arial, Verdana
  • Font-Style: normal, italic, oblique (leaning; not well supported)
  • Font-Size: Absolute size (default is 16px) or relative size such as 'large', 'small', 'smaller' (should use relative)
  • Font-Variant: normal or small-caps
  • Font-Weight: bold, bolder, normal, lighter
